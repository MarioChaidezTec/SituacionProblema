# Sistema de Gestion de Envios 
Sistema de Registro de envios desarrollado en C++
## Descripcion 
El programa consiste en un sistema de gestión de envíos. Se crea un envío el cual se le asigna productos desde un .csv, un repartidor y un cliente. El programa es capaz de crear un envío, agregarle productos, eliminarle productos, mostrar su información, cambiarle el estado y asignarle un repartidor. 
## Caracteristicas 
+ Gestion de envios
+ Asignacion de repartidores a diferentes envios
+ Eliminacion de Productos
+ Mostrar informacion de clientes
## Tecnologias 
+ Lenguaje de programacion: C++
+ IDE Utilizado: Clion
+ Sistema de Build: CMake
## Instalacion 
+ Descargar el ejecutable
+ Tener ficheros en formato .csv con listas de productos en formato (Nombre, precio unitario, cantidad)
